# Pages Drop‑in (Static)
קבצים מינימליים לעדכון GitHub Pages **רק ע"י הוספת קבצים** — ללא שרת.

## מה מעלים לריפו
- `index.html`
- `styles.css`
- `app.js`

אם יש כבר `index.html` ישן בשורש — ההעלאה דרך GitHub תבצע **Replace** אוטומטי לקובץ (עדיין פעולה של "הוספה").

## מה מקבלים
- הרשמה/כניסה (לוקאלי via `localStorage`)
- בחירת יחידות 3/4/5 ושמירה למשתמש
- קטלוג תרגול בסיסי (MCQ/Cloze) שעובד על Pages

### שדרוג עתידי
אפשר להחליף את ה־localStorage ב־API אמיתי כשיהיה שרת חי, ע"י שינוי הקריאות ב־`app.js`.
